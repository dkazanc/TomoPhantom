Use Artefacts Simulator

Use Artefacts Simulator#

TomoPhantom comes with an artefacts simulator that can add a variety of imaging inaccuracies and noise, therefore aiming to emulate real imaging artefacts. While more detailed information needs to be added here, please see the API for tomophantom.artefacts().
