Use Phantom Libraries#

TomoPhantom provides a number of phantoms for users that can be used out-of-the-box. There are static (2D, 3D) or dynamic (2D+time, 3D+time) phantoms with the parameters for them stored in Library files.

The users can generate phantoms by specifying a Model number from the associated library file. Alternatively, more experienced user can build a phantom using Object API.
